# Number Of Island_BFS

- 2차원 배열이 있을때, 그 행렬에서 육지의 갯수를 구하는 문제.

> 1. Input 
> {'1','1','1','0','1'},
> {'1','1','0','0','0'},
> {'1','1','0','0','1'},
> {'0','0','0','0','1'}
> 1은 육지, 0은 바다
> 2. Output : 3

> **BSF(너비 우선 탐색)**
> BFS를 이용한 문제이며, Queue에 육지의 포인터(x,y)의 배열을 담아서 반복적으로 Search한다.
> 주로 BFS 문제는 Queue를 사용한다.
